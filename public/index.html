⚙️ LLM 配置

配置 OpenAI 兼容的 LLM API，用于智能修复建议和屏幕阅读器路径分析

支持 OpenAI 兼容的 API 端点，如 Ollama、LM Studio、火山方舟等

💡 常用配置示例

OpenAI

Base URL: https://api.openai.com/v1

Model: gpt-3.5-turbo, gpt-4, gpt-4o

Ollama (本地)

Base URL: http://localhost:11434/v1

Model: llama3, mistral, codellama 等

火山方舟

Base URL: https://ark.cn-beijing.volces.com/api/v3

Model: 接入点 ID (如 ep-xxx)

LM Studio (本地)

Base URL: http://localhost:1234/v1

Model: 已加载的模型名称
